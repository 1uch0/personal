RAG-Based Industrial Copilot for Adaptive Manufacturing Systems in Industry 4.0

This system uses the APIs of GPT-4, Gemini, and Anthropic, together with JSON-stored modular manufacturing resources. A Retrieval-Augmented Generation (RAG) pipeline supplies the modular system, while the user interface provides flexible access to recipes, resources, historical data, and related information.

Large Language Models to support Altruistic Collaborative Healing in Smart Manufacturing

This project uses AI language models (like ChatGPT) to help manufacturing machines collaborate like a team—when one machine runs low on energy or materials, nearby "donor" machines step in to share resources through natural-language negotiation, making the whole system more resilient and adaptive. We tested this with autonomous robots in a simulated factory, and the AI-mediated approach improved reliability by 80%, showing promise for smarter, self-healing factories of the future.

An Access Control Method Against Unauthorized and Noncompliant Behaviors Leveraging Large Language Models

The project enhances Behavior-Based Access Control (BBAC) by integrating Large Language Models (LLMs) to interpret policies, analyze user behavior, and make context-aware, real-time permission decisions. This approach improves detection of suspicious activities, balances security with workflow efficiency, and has shown promising accuracy and effectiveness in early synthetic data experiments.

Implementing Altruistic Collaborative Healing in Smart Manufacturing Context, Simulation and Analysis

This project develops “altruistic collaborative healing” for manufacturing, enabling elements to share resources in real time to maintain operations. It builds on biological principles, uses ROS2 robotic simulations for implementation, and validates energy-sharing scenarios that reduce downtime compared to manual methods.
